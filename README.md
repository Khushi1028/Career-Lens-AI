# Career Lens AI

An AI-powered platform for MCA students to analyze resumes, check ATS scores, and get skill gap recommendations.

## 🚀 Features
- **Resume Checker**: Upload resume and get ATS match score
- **Skill Distribution Analytics**: See skill strengths and gaps
- **Automated Text Parsing**: Extracts key info from resume
- **Course Recommendations**: Suggestions based on missing skills
- **Dynamic Dashboard**: Visual analytics and progress tracking

## 🛠️ Tech Stack
- **Backend**: Flask, Python
- **ML**: Scikit-learn, Ensemble Learning
- **Frontend**: HTML, CSS, JavaScript
- **Database**: SQLite / CSV
